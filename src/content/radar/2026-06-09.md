## 🌐 Web / Frameworks / Languages

- **React Compiler v1.0** — stable ออกจริง **7 ต.ค. 2025** [official], optimize component/hook อัตโนมัติ ใช้ production ที่ Meta แล้ว ([react.dev — React Compiler 1.0](https://react.dev/blog/2025/10/07/react-compiler-1))
- **React Compiler "Rust edition"** — ทีม React port logic ส่วนใหญ่จาก TypeScript → Rust ด้วย AI, สถานะ "coming soon" ([Joe Savona บน X](https://x.com/en_JS/status/2031606726689173846)); เลข **"เร็วขึ้น ~10x"** มาจาก blog ภายนอก ⚠️ ([Stork.AI](https://www.stork.ai/blog/reacts-rust-rewrite-just-killed-manual-hooks))
- **TypeScript 7.0** (เป้า กลางปี 2026) — compiler เขียนด้วย **Go**, strict-by-default, ตัด ES5 target + AMD/UMD/SystemJS, type-check parity ~99.6% ([State of TypeScript 2026](https://devnewsletter.com/p/state-of-typescript-2026/))
- **rolldown** (Rust bundler) กำลังเป็น default ของ Vite ecosystem (Nuxt/SvelteKit/Astro) → แทน esbuild/Rollup; บางทีม build เร็วขึ้น 2.5 นาที → 40 วิ ⚠️ ([State of JS 2026](https://devnewsletter.com/p/state-of-javascript-2026/))

→ **ภาพรวม:** Rust ยึดชั้น tooling ที่ต้องการ performance, TS ยังเป็นภาษาหลักของ application
