## ☸️ Cloud / DevOps / Infra

**Kubernetes 1.37 ประกาศ feature freeze, release เมื่อ 26 ส.ค. 2026** — Production Readiness Freeze ผ่านมาแล้ว (9 มิ.ย. AoE / 10 มิ.ย. UTC), Enhancements Freeze 16–17 มิ.ย., release target 26 ส.ค. 2026 ขณะนี้ได้เข้าโหมดสำหรับ bug fix + critical patch เท่านั้น ([Kubernetes.dev](https://www.kubernetes.dev/resources/release/)) [official]

**K8s 1.36 เสถียร: GPU scheduling + fine-grained authz** — Native GPU scheduling ผ่าน Workload Aware Scheduling (WAS) + Dynamic Resource Allocation (DRA) ถึง beta; fine-grained Kubelet authorization stable; สำคัญต่อ ML workloads ([Kubernetes.io/releases](https://kubernetes.io/releases/)) [official]

**KubeCon India 18–19 มิ.ย. 2026 บัดนี้ผ่านแล้ว** — งานใหญ่สมการ release cycle; OpenTelemetry หลบออกจาก CNCF sandbox เป็น graduated, Kyverno 1.18 เป็น release แรกหลังทำตัวเป็น CNCF project ([CNCF news](https://www.cncf.io/news/)) [official]

→ **งาน DRA + Kyverno security:** ทำให้ GPU/TPU ใช้ได้จริงใน shared clusters, policy-as-code ป้องกันเก็บ secret ในรูป

## 🌐 Web / Frameworks / Languages

**React Compiler จาก TS → Rust เสร็จ 9 มิ.ย., รวมใน Next.js 16.4** — PR #36173 merge ยอดเรื่องโปรแกรม Compiler logic ทั้งชุดจาก TypeScript ไปเป็น Rust ด้วย AI; Vercel ยืนยันว่า Next.js 16.4 build เร็วขึ้น **40%+** end-to-end ([React Compiler in Rust](https://byteiota.com/react-compiler-rust-nextjs-16-4-faster-builds/)) [official]

**ไม่แทน TS แต่แทน compiler tooling** — AST → HIR → CFG/SSA → optimization เดิมทั้งหมด แค่เขียนด้วย Rust arena-based structures ไม่ใช่ object graph; ยังคงเขียน components ใน JS/TS ([React Compiler in Rust](https://blog.nidhin.dev/react-compiler-in-rust)) [official]

**Rust ยึดสายเครื่องมือ:** SWC (Babel), Turbopack (Webpack), Rolldown (Rollup), Biome (ESLint) — ทั้งหมดเข้ามา Vite 8, Next.js 16+ ลัดสายของ pure JS/TS toolchain เพื่อเรียวความเร็ว ([Best Rust Tools](https://cssauthor.com/best-rust-tools-for-javascript-developers/)) ⚠️

→ **ภาพรวม:** Node/TypeScript อยู่ layer application, Rust ทำให้ build/parse/transform รวดเร็ว — โมเดลสหาย ecosystem
