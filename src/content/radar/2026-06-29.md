## ☸️ Cloud / DevOps / Infra

- **Kubernetes AI/GPU workloads เป็นมาตรฐาน** — ผลสำรวจ 500+ infrastructure leaders: 93% ใช้ K8s แล้ว, 80% ใน production, 84% คาดว่า 5 ปีหน้าจะสร้าง 50% ของแอปปลิเคชันใหม่บน Kubernetes ([portworx.com — enterprise adoption trends](https://portworx.com/blog/kubernetes-enterprise-adoption-trends/))
- **GPU scheduling และ advanced orchestration เป็นของบังคับ** — Over 90% ของทีม expect AI workloads จะเพิ่มขึ้นบน K8s ต้องการ orchestration ที่ดีเพื่อจัดการ expensive GPUs และ real-time inference services ([devzero.io — K8s optimization](https://www.devzero.io/blog/top-kubernetes-infrastructure-optimization-tools-for-2026))
- **Docker Kanvas ท้าชิง Helm/Kustomize** — Docker เปิดตัว Kanvas เพื่อเชื่อมต่อ local development กับ cloud production โดยแปลง Docker Compose เป็น Kubernetes artifacts โดยอัตโนมัติ ([infoq.com — Docker Kanvas](https://www.infoq.com/news/2026/01/docker-kanvas-cloud-deployment/))

→ **เกี่ยวกับงาน tune-keycloak:** GPU scheduling และ FinOps tools ควรทำให้ AI workloads สามารถนำไปใช้ได้จริง

## 🌐 Web / Frameworks / Languages

- **React Compiler เปลี่ยนเป็น Rust ได้ 40% เร็วขึ้น** — PR merge 9 มิ.ย. 2026: Vercel confirms Rust version ships ใน Next.js 16.4, drop-in Babel plugin เร็วกว่า TypeScript original 3 เท่า, linked เข้า Turbopack ได้ 40% faster compilation end-to-end ([byteiota.com — Next.js 16.4](https://byteiota.com/react-compiler-rust-nextjs-16-4-faster-builds/))
- **JavaScript toolchain ฝ่าย native languages ไป** — เครื่องมือ build ต่างๆ (SWC, OXC, React Compiler) ต่างเขียนเป็น Rust หรือภาษาที่ compile เพื่อให้ 10-20x speed improvements แต่ developers ยังเขียน React components เป็น TS/JS ตามปกติ ([stork.ai — React Compiler Rust](https://www.stork.ai/blog/reacts-rust-rewrite-just-killed-manual-hooks))
- **Rust WASM production-ready** — Figma, 1Password, Shopify ต่างใช้ Rust WASM แล้ว, tooling (wasm-bindgen) สำหรับ WebAssembly integration เหมาะสมและได้ performance benefits จริงๆ ([nandann.com — Rust WASM 2026](https://www.nandann.com/blog/rust-wasm-production-2026))

→ **ภาพรวม:** JavaScript ecosystem กำลังกลายเป็น polyglot ที่ core layer ใช้ Rust/compiled languages สำหรับ performance
