## 🌐 Web / Frameworks / Languages

- **Rust web frameworks — Actix Web, Axum, Rocket ขึ้นเป็น backend standard** [official] — สำหรับ production ที่ต้องการ memory safety + near-C performance ไม่มี garbage collection overhead ปัจจุบัน community เลือก Actix/Axum (ลักษณะเหมือน Express/Fastify) หรือ Leptos สำหรับ full-stack ([Rust Framework Review 2026](https://medium.com/@priya.raimagiya/the-5-best-rust-frontend-frameworks-reviewed-for-2026-4694dad8f181))

- **Yew — Rust frontend framework ที่ใกล้เคียง React** [official] — 30,000+ GitHub stars, 4M+ downloads crates.io mental model transfer จาก React ตรงๆ ลองใช้ได้สำหรับ WebAssembly frontend ที่ต้องประสิทธิภาพสูง ([Yew Project](https://libs.tech/rust/web-frameworks))

- **Next.js + React Server Components — dominates job market** [official] — server-side rendering pioneer ในecosystem JS/TS ยังมี momentum ปัจจุบัน modern web dev focus ที่ instant feedback (HMR), type safety (TypeScript default), integrated testing, deployment automation ([Next.js 2026 Status](https://wasp.sh/resources/2026/02/24/best-frameworks-web-dev-2026))

→ **ภาพรวม:** Rust web frameworks กำลัง serious ในอุตสาหกรรม JavaScript/TypeScript ยังครอบ application layer; trend คือ split tools ตามลักษณะ (performance → Rust, business logic → TS)
