## 🌐 Web / Frameworks / Languages

- **Rolldown bundler มี speed 10-30x ของ Rollup** — Rust bundler ใหม่กำลังเป็น default ของ Vite ecosystem (Nuxt, SvelteKit, Astro) ความเร็วอาจเร็วขึ้นกว่า Rollup 10-30 เท่า ทำให้ build process หลายทีมเร็วขึ้นจากหลายนาทีเหลือเพียงสิบวินาที ⚠️ ([Turbopack 2026 Guide](https://pockit.tools/blog/turbopack-nextjs-bundler-complete-guide/))
- **React Compiler rewrite เป็น Rust ได้ 10x ผลงาน** — Meta ทีม React port logic ส่วนใหญ่จาก TypeScript → Rust สร้างการเปลี่ยนแปลง 10x ที่เร็วขึ้น ผลเบิกต่อการใช้: 12% faster initial loads, 2.5x faster interactions บน Instagram และ Meta Quest Store [official] ([React Compiler Rust](https://www.stork.ai/blog/reacts-rust-rewrite-just-killed-manual-hooks))
- **TypeScript compiler เขียนด้วย Go 10x เร็ว** — ทีม TypeScript port compiler ไปยัง Go ทำให้บิลด์เร็วขึ้น 10 เท่า รวมกับ rolldown ผลรวมอาจเห็นการลดเวลาบิลด์ 38-64% ⚠️ ([Turbopack 2026 Guide](https://pockit.tools/blog/turbopack-nextjs-bundler-complete-guide/))

→ **สรุป:** Rust/Go rewrites แทนที่ JavaScript tooling ลดเวลาบิลด์อย่างมีนัยสำคัญ
